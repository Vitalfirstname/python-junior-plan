Учебный план: Python Junior

Формат: академический курс на 140 дней
Целевая аудитория: взрослые гуманитарии, начинающие программисты
Цель: уверенный уровень Junior Python-разработчика

I. Введение в Python (7 дней)
- Что такое Python и зачем он нужен
- Установка Python, работа с интерпретатором
- Hello, World! и запуск скриптов
- Переменные, типы данных (int, float, str, bool)
- Ввод/вывод данных

II. Структуры данных (14 дней)
- Списки (list): создание, доступ, методы
- Кортежи (tuple), множества (set), словари (dict)
- Индексация, срезы, вложенные структуры
- Преобразование типов

III. Управляющие конструкции (10 дней)
- Условные операторы: if, elif, else
- Циклы: while, for
- Вложенные циклы
- break, continue, pass

IV. Функции (14 дней)
- Создание функций: def
- Аргументы, параметры, return
- Область видимости переменных
- Lambda-функции, встроенные функции

V. Работа с файлами (10 дней)
- Чтение и запись файлов
- Контекстный менеджер with
- Обработка текстов

VI. Исключения (7 дней)
- Обработка ошибок try/except
- finally, else
- Создание собственных исключений

VII. Модули и пакеты (10 дней)
- Импорт модулей
- Стандартные библиотеки
- Установка сторонних библиотек (pip)

VIII. ООП — основы (14 дней)
- Классы и объекты
- Конструкторы и методы
- Наследование, инкапсуляция

IX. Практика: мини-проекты (28 дней)
- Калькулятор
- Чек-лист (to-do list)
- Работа с API
- Парсинг сайта
- Простая игра

X. Итоговый проект (14 дней)
- Планирование
- Разработка
- Презентация проекта

Дополнительно:
- Работа с Git и GitHub
- Работа в PyCharm
- Основы командной строки
- Советы по поиску работы и собеседованию
